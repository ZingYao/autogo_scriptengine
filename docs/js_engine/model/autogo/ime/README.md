# ime 模块

## 模块简介

ime 模块提供了输入法相关的功能，包括剪贴板操作、文本输入、输入法切换等。

## 方法列表

### ime.getClipText
获取剪切板内容

**使用示例：**
```javascript
// 调用 ime.getClipText 方法
ime.getClipText();
```

---

### ime.setClipText
设置剪切板内容

**使用示例：**
```javascript
// 调用 ime.setClipText 方法
ime.setClipText();
```

---

### ime.keyAction
模拟按键

**使用示例：**
```javascript
// 调用 ime.keyAction 方法
ime.keyAction();
```

---

### ime.inputText
输入文本

**使用示例：**
```javascript
// 调用 ime.inputText 方法
ime.inputText();
```

---

### ime.getIMEList
获取输入法列表

**使用示例：**
```javascript
// 调用 ime.getIMEList 方法
ime.getIMEList();
```

---

### ime.setCurrentIME
设置当前输入法

**使用示例：**
```javascript
// 调用 ime.setCurrentIME 方法
ime.setCurrentIME();
```
