# yolo 模块

## 模块简介

yolo 模块提供了 YOLO（You Only Look Once）目标检测功能，用于实时对象检测。

## 方法列表

### yolo.new
创建一个新的YOLO实例

**使用示例：**
```javascript
// 调用 yolo.new 方法
yolo.new();
```

---

### yolo.detect
检测屏幕上的对象

**使用示例：**
```javascript
// 调用 yolo.detect 方法
yolo.detect();
```

---

### yolo.detectFromImage
检测图片中的对象

**使用示例：**
```javascript
// 调用 yolo.detectFromImage 方法
yolo.detectFromImage();
```

---

### yolo.detectFromBase64
检测Base64图片中的对象

**使用示例：**
```javascript
// 调用 yolo.detectFromBase64 方法
yolo.detectFromBase64();
```

---

### yolo.detectFromPath
检测文件图片中的对象

**使用示例：**
```javascript
// 调用 yolo.detectFromPath 方法
yolo.detectFromPath();
```

---

### yolo.close
关闭YOLO实例

**使用示例：**
```javascript
// 调用 yolo.close 方法
yolo.close();
```
