# plugin 模块

## 模块简介

plugin 模块提供了加载外部 APK 插件的功能，允许动态加载和执行外部代码。

## 方法列表

### plugin.loadApk
加载外部APK

**使用示例：**
```javascript
// 调用 plugin.loadApk 方法
plugin.loadApk();
```

---

### plugin.newContext
创建Context参数

**使用示例：**
```javascript
// 调用 plugin.newContext 方法
plugin.newContext();
```

---

### plugin.newAssetManager
创建AssetManager参数

**使用示例：**
```javascript
// 调用 plugin.newAssetManager 方法
plugin.newAssetManager();
```
