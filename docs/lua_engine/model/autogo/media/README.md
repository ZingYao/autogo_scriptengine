# media 模块

## 模块简介

media 模块提供了媒体相关的功能，包括音频播放、文件扫描、短信发送等。

## 方法列表

### media.scanFile
扫描路径path的媒体文件

**使用示例：**
```lua
-- 调用 media.scanFile 方法
media.scanFile();
```

---

### media.playMP3
播放MP3文件

**使用示例：**
```lua
-- 调用 media.playMP3 方法
media.playMP3();
```

---

### media.sendSMS
发送短信

**使用示例：**
```lua
-- 调用 media.sendSMS 方法
media.sendSMS();
```
