# hud 模块

## 模块简介

hud 模块提供了在屏幕上显示浮动窗口（HUD）的功能。可以创建自定义的浮动界面，用于显示调试信息、状态提示等内容。

## 方法列表

### hud.new
创建一个新的HUD对象

**使用示例：**
```lua
-- 调用 hud.new 方法
hud.new();
```

---

### hud.setPosition
设置HUD的显示位置

**使用示例：**
```lua
-- 调用 hud.setPosition 方法
hud.setPosition();
```

---

### hud.setBackgroundColor
设置HUD的背景颜色

**使用示例：**
```lua
-- 调用 hud.setBackgroundColor 方法
hud.setBackgroundColor();
```

---

### hud.setTextSize
设置HUD中文本的字体大小

**使用示例：**
```lua
-- 调用 hud.setTextSize 方法
hud.setTextSize();
```

---

### hud.setText
设置HUD中要显示的文本内容

**使用示例：**
```lua
-- 调用 hud.setText 方法
hud.setText();
```

---

### hud.show
显示HUD

**使用示例：**
```lua
-- 调用 hud.show 方法
hud.show();
```

---

### hud.hide
隐藏HUD

**使用示例：**
```lua
-- 调用 hud.hide 方法
hud.hide();
```

---

### hud.isVisible
返回HUD是否可见

**使用示例：**
```lua
-- 调用 hud.isVisible 方法
hud.isVisible();
```

---

### hud.destroy
销毁HUD对象，释放资源

**使用示例：**
```lua
-- 调用 hud.destroy 方法
hud.destroy();
```
