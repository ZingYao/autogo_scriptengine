# motion 模块

## 模块简介

motion 模块提供了模拟用户操作的功能，包括点击、滑动、按键等手势操作。

## 方法列表

### motion.touchDown
按下屏幕

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| x | int | 是 | 屏幕X坐标 |
| y | int | 是 | 屏幕Y坐标 |
| fingerID | int | 否 | 手指ID，默认为0 |
| displayId | int | 否 | 显示器ID，默认为0 |

**返回值：**

无

**使用示例：**
```lua
-- 在坐标(500, 500)处按下屏幕
motion.touchDown(500, 500);

-- 使用指定手指ID按下
motion.touchDown(500, 500, 1);

-- 使用指定显示器ID按下
motion.touchDown(500, 500, 0, 1);
```

### motion.longClick
长按

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| x | int | 是 | 屏幕X坐标 |
| y | int | 是 | 屏幕Y坐标 |
| duration | int | 否 | 长按持续时间（毫秒），默认为500 |
| fingerID | int | 否 | 手指ID，默认为0 |
| displayId | int | 否 | 显示器ID，默认为0 |

**返回值：**

无

**使用示例：**
```lua
-- 在坐标(500, 500)处长按（默认500毫秒）
motion.longClick(500, 500);

-- 长按1000毫秒

-- 使用指定手指ID长按
motion.longClick(500, 500, 500, 1);

-- 在指定显示器上长按
motion.longClick(500, 500, 500, 0, 1);
```

### motion.swipe
滑动

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| x1 | int | 是 | 起始点X坐标 |
| y1 | int | 是 | 起始点Y坐标 |
| x2 | int | 是 | 结束点X坐标 |
| y2 | int | 是 | 结束点Y坐标 |
| duration | int | 是 | 滑动持续时间（毫秒） |
| fingerID | int | 否 | 手指ID，默认为0 |
| displayId | int | 否 | 显示器ID，默认为0 |

**返回值：**

无

**使用示例：**
```lua
-- 从(100, 500)滑动到(900, 500)，耗时500毫秒
motion.swipe(100, 500, 900, 500, 500);

-- 使用指定手指ID滑动
motion.swipe(100, 500, 900, 500, 500, 1);

-- 在指定显示器上滑动
motion.swipe(100, 500, 900, 500, 500, 0, 1);
```
