# uiacc 模块

## 模块简介

uiacc 模块提供了无障碍服务（Accessibility）的功能，用于查找和操作界面元素。

## 方法列表

**使用示例：**
```lua
-- 调用 uiacc.new 方法
uiacc.new();
```
