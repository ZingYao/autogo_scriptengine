# system 模块

## 模块简介

system 模块提供了系统级别的功能，包括进程管理、内存监控、系统设置等。

## 方法列表

### system.getPid
获取进程ID

**使用示例：**
```lua
-- 调用 system.getPid 方法
system.getPid();
```

---

### system.getMemoryUsage
获取内存使用

**使用示例：**
```lua
-- 调用 system.getMemoryUsage 方法
system.getMemoryUsage();
```

---

### system.getCpuUsage
获取CPU使用率

**使用示例：**
```lua
-- 调用 system.getCpuUsage 方法
system.getCpuUsage();
```

---

### system.restartSelf
重启自身

**使用示例：**
```lua
-- 调用 system.restartSelf 方法
system.restartSelf();
```

---

### system.setBootStart
设置开机自启

**使用示例：**
```lua
-- 调用 system.setBootStart 方法
system.setBootStart();
```
