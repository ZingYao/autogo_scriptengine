# lfs 模块

## 模块简介

lfs 模块提供了相关的功能支持。

## 方法列表

### lfs.attributes
获取文件或目录属性

**使用示例：**
```lua
-- 调用 lfs.attributes 方法
lfs.attributes();
```

---

### lfs.chdir
改变当前工作目录

**使用示例：**
```lua
-- 调用 lfs.chdir 方法
lfs.chdir();
```

---

### lfs.currentdir
获取当前工作目录

**使用示例：**
```lua
-- 调用 lfs.currentdir 方法
lfs.currentdir();
```

---

### lfs.mkdir
创建目录

**使用示例：**
```lua
-- 调用 lfs.mkdir 方法
lfs.mkdir();
```

---

### lfs.rmdir
删除空目录

**使用示例：**
```lua
-- 调用 lfs.rmdir 方法
lfs.rmdir();
```

---

### lfs.touch
更新文件时间

**使用示例：**
```lua
-- 调用 lfs.touch 方法
lfs.touch();
```
