# math 模块

## 模块简介

math 模块提供了相关的功能支持。

## 方法列表

### math.tointeger
将值转换为整数

**使用示例：**
```lua
-- 调用 math.tointeger 方法
math.tointeger();
```

---

### math.type
获取数字的类型

**使用示例：**
```lua
-- 调用 math.type 方法
math.type();
```

---

### math.ult
无符号比较两个整数

**使用示例：**
```lua
-- 调用 math.ult 方法
math.ult();
```
