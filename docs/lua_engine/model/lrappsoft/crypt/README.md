# crypt 模块

## 模块简介

crypt 模块提供了相关的功能支持。

## 方法列表

### crypt.aes_crypt
AES 加密/解密

**使用示例：**
```lua
-- 调用 crypt.aes_crypt 方法
crypt.aes_crypt();
```

---

### crypt.aes_keygen
生成 AES 密钥

**使用示例：**
```lua
-- 调用 crypt.aes_keygen 方法
crypt.aes_keygen();
```

---

### crypt.aes_ivgen
生成随机 IV

**使用示例：**
```lua
-- 调用 crypt.aes_ivgen 方法
crypt.aes_ivgen();
```

---

### crypt.rsa_generate_key
生成 RSA 密钥对

**使用示例：**
```lua
-- 调用 crypt.rsa_generate_key 方法
crypt.rsa_generate_key();
```

---

### crypt.rsa_encrypt
RSA 加密

**使用示例：**
```lua
-- 调用 crypt.rsa_encrypt 方法
crypt.rsa_encrypt();
```

---

### crypt.rsa_decrypt
RSA 解密

**使用示例：**
```lua
-- 调用 crypt.rsa_decrypt 方法
crypt.rsa_decrypt();
```
