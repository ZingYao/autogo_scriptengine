# console 模块

## 模块简介

console 模块提供了控制台窗口的功能，用于显示日志和调试信息。

## 方法列表

### console.show
显示控制台悬浮窗

**使用示例：**
```lua
-- 调用 console.show 方法
console.show();
```

---

### console.showTitle
显示或者隐藏控制台标题栏

**使用示例：**
```lua
-- 调用 console.showTitle 方法
console.showTitle();
```

---

### console.lockConsole
锁定控制台窗口

**使用示例：**
```lua
-- 调用 console.lockConsole 方法
console.lockConsole();
```

---

### console.unlockConsole
解除锁定控制台窗口

**使用示例：**
```lua
-- 调用 console.unlockConsole 方法
console.unlockConsole();
```

---

### console.dismiss
关闭控制台窗口

**使用示例：**
```lua
-- 调用 console.dismiss 方法
console.dismiss();
```

---

### console.setPos
设置控制台窗口的位置和大小

**使用示例：**
```lua
-- 调用 console.setPos 方法
console.setPos();
```

---

### console.println
打印日志到控制台窗口

**使用示例：**
```lua
-- 调用 console.println 方法
console.println();
```

---

### console.clearLog
清除日志

**使用示例：**
```lua
-- 调用 console.clearLog 方法
console.clearLog();
```

---

### console.setTitle
设置控制台标题

**使用示例：**
```lua
-- 调用 console.setTitle 方法
console.setTitle();
```

---
